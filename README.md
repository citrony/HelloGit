## 共同作業を始めるには
1. 招待側が setting=>Collaborators で被招待者を招待する
以降は被招待者側の操作
2. git clone https://github.com/citrony/HelloGit
3. git checkout -b 新しく作るブランチ名
4. git push -u origin 作ったブランチ名
5. ファイルを編集する
6. git add .
7. git commit -m"コミットメッセージ"
8. git push origin 作ったブランチ名
